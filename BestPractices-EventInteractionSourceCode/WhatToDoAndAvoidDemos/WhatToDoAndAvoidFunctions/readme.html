Deploy this with the ARM template and it will configure all of the settings for you. It will deploy 4x websites in 2x hosting plans across 2x regions with a couple of redis caches, not all of this is needed. Make sure you update the hard coded date in GetQuestions and the javascript code on the main site to match, otherwise you will not get any questions showing on the list. The CSV files include uploadable sample data, use Microsoft Storage Explorer to upload them into your table storage account (the one ending in nstacc) to allow the page to render something.
